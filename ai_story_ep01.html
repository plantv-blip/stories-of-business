&circled1;
자본조달 — 누구의 돈으로 만드는가
BS 우측 · WACC · 자본구조 · DuPont 레버리지
OpenAI

Microsoft 단일 축 의존 구조. Microsoft로부터 누적 $13B+ 투자를 받았고, PBC 전환 후 27% 지분을 부여했다. Azure가 독점 클라우드 인프라이며, 매출의 상당 부분이 Microsoft 생태계(Copilot, Teams)를 통해 발생한다.

장점은 단일 파트너의 전폭적 자원 지원. 리스크는 Microsoft의 전략 변경 시 사업 모델 전체가 흔들릴 수 있는 의존 구조.

누적 투자: $13B+ (Microsoft) · 총 밸류에이션: $300B
Anthropic

다자 균형 + 미션 보호 장치. Amazon $8B, Google $3B+, VC 다수(Lightspeed, Fidelity, Iconiq)에서 자본을 분산 유치했다. LTBT(장기이익신탁)가 이사 과반 선임권을 가져 어느 투자자도 단독 지배력을 행사할 수 없다.

장점은 자본 독립성과 미션 보호. 리스크는 다수 투자자 간 이해 조율의 복잡성, 그리고 IPO 압력 증가.

누적 투자: $8B(Amazon) + $3B+(Google) + VC · 밸류에이션: $380B
Node 1 통찰

두 회사의 자본 구조는 CEO의 세계관을 반영한다. Altman은 “하나의 강력한 파트너와 올인”을 택했고, Amodei는 “아무도 지배하지 못하는 구조”를 설계했다. Microsoft 의존도가 높은 OpenAI는 빠르게 자원을 동원할 수 있지만, Anthropic의 분산 구조는 장기적 독립성을 보장한다. DuPont 분석에서 레버리지 요소가 다른 셈이다.

&circled2;
자산 · 투자 — GPU를 소유할 것인가, 빌릴 것인가
BS 좌측 · 자산회전율 · ROIC · NWC
OpenAI

인프라 자체 소유를 향한 대규모 투자. 2030년까지 $600B 규모의 컴퓨트 인프라 투자를 선언했다. CoreWeave($22.4B), Oracle($300B), AWS($38B)와 장기 계약을 체결하고, 자체 데이터센터 건설에도 착수했다. 컴퓨팅 용량이 2024년 0.6GW에서 2025년 1.9GW로 3배 증가.

인프라 투자: $600B 목표(~2030) · 컴퓨트: 1.9GW(2025)
Anthropic

파트너 인프라 위에서의 효율 극대화. 자체 데이터센터 대신 AWS 인프라 위에서 커널 레벨 최적화를 수행한다. Amazon의 $8B 투자 자체가 AWS 사용 계약과 연동되어 있어, 인프라 비용을 투자로 전환하는 구조. 자산을 가볍게 유지하면서 기술적 깊이로 효율을 높이는 전략.

인프라: AWS 기반 · Amazon $8B = 인프라 크레딧 성격
Node 2 통찰

반도체 산업에서 삼성(팹 자체 소유)과 팹리스(TSMC에 위탁)의 구조 차이와 유사하다. OpenAI는 인프라 소유를, Anthropic은 인프라 효율을 택했다. 자산회전율(Asset Turnover) 관점에서, Anthropic이 같은 매출 대비 훨씬 가벼운 자산 구조를 유지하고 있다. 이것이 밸류에이션 프리미엄의 한 원인이다.

&circled3;
밸류체인 — 비용은 어디서 발생하는가
P&L 비용 · Porter Value Chain · 원가구조 · Overhead
OpenAI

$9B 적자의 구조: 훈련 + 추론 + 소비자 마케팅. AI 모델 훈련 비용(GPU 사용), 800M 사용자의 추론 비용(사용할 때마다 GPU 작동), 그리고 소비자 확보를 위한 마케팅·제품 개발 비용이 삼중으로 발생한다. 무료 티어와 광고 티어 추가는 사용자 확장에 기여하지만 추론 비용을 늘린다.

연간 적자: ~$9B · 적자율: 매출의 ~45%
Anthropic

$3B 적자의 구조: 훈련 + R&D 집중. 소비자 마케팅 비용이 거의 없고(매출의 85%가 API = B2B), 비용의 대부분이 모델 훈련과 안전 연구에 집중된다. 엔터프라이즈 API 고객은 사용량에 따라 과금되므로, 추론 비용이 매출과 직접 연동되는 구조.

연간 적자: ~$3B · 적자율: 매출의 ~33% · 2028 BEP 목표
Node 3 통찰

Porter 관점에서 핵심 차이는 “Overhead 구조”다. OpenAI는 소비자 800M명을 대상으로 무료 추론을 제공하는 비용이 고정 Overhead로 작용한다. Anthropic은 API 종량제이므로 비용이 매출에 비례하는 변동비 구조다. 같은 적자이지만 OpenAI의 적자는 “고정비 기반”이고 Anthropic의 적자는 “투자 기반”이다 — 시장이 후자에 더 높은 프리미엄을 주는 이유다.

&circled4;
매출 · 수익 — 누구에게, 어떻게 파는가
P&L 상단 · Pricing Power · Porter 5 Forces
OpenAI

소비자 구독 중심 + 빠른 확장. ChatGPT Plus $20/월 구독이 매출 핵심. 800M 주간 사용자 중 유료 전환율이 수익성의 열쇠. 엔터프라이즈(ChatGPT Enterprise/Team)도 성장 중이지만 아직 매출의 30%. 2025년 무료 광고 티어를 추가하며 더 넓은 저변 확보 시도.

성장 속도: ARR $6B(2024말) → $20B(2025말). 연 233% 성장.

ARPU: ~$20/월 · B2C:B2B = 70:30 · 유료 구독자: 3M+(기업)
Anthropic

엔터프라이즈 API 중심 + 폭발적 성장률. Claude API의 토큰 종량 과금이 매출의 85%. 엔터프라이즈 시장 점유율 32%(2025)로 1위. 코드 생성 분야에서 42% 점유율로 OpenAI(21%)를 2배 차이로 앞선다.

성장 속도: ARR $1B(2024말) → $9B(2025말) → $19B(2026.03). 연 900% 성장.

B2B:B2C = 85:15 · 엔터프라이즈 고객: 300K+ · 코드 점유율: 42%
Node 4 통찰

Porter 5 Forces에서 가장 결정적인 차이는 전환 비용(Switching Cost)이다. OpenAI의 소비자 사용자는 언제든 Claude나 Gemini로 전환할 수 있다(낮은 전환 비용). 하지만 Anthropic의 API 고객은 자사 제품에 Claude를 깊이 통합해서 사용하므로 전환 비용이 높다. 사용자 수는 OpenAI가 압도적이지만, “고객 1명당 매출 안정성”은 Anthropic이 구조적으로 우위다.

&circled5;
이익 · EVA — 가치를 창출하고 있는가
P&L 하단 · EVA=NOPAT−WACC×투하자본 · ROIC Spread
OpenAI

매출 $20B, 적자 $9B — 수익성 미증명. 매출의 45%를 적자로 소진하고 있다. $600B 인프라 투자 계획은 향후 수년간 추가 자본 소진을 예고한다. EVA 관점에서 투하자본 대비 경제적 이익은 아직 음(-)이다. IPO를 통한 공개 시장 검증이 수익성 전환의 압력이 될 전망.

매출: $20B · 적자: $9B · 적자율: ~45% · EVA: 음(-)
Anthropic

매출 $9B, 적자 $3B — 2028 BEP를 설계 중. 매출의 33%를 적자로 소진하지만, 2028년 손익분기점을 명시적 목표로 설정하고 있다. API 종량제 구조상 매출 성장이 곧 마진 개선과 연결되는 경로가 존재한다. 적자의 대부분이 R&D이므로, 연구 효율이 개선되면 급격한 이익 전환 가능성.

매출: $9B · 적자: $3B · 적자율: ~33% · BEP: 2028 목표
Node 5 통찰

두 회사 모두 아직 EVA가 음이다 — 투하자본의 기회비용을 넘는 이익을 내지 못하고 있다. 하지만 시장이 Anthropic에 더 높은 밸류에이션($380B vs $300B)을 부여하는 이유는 적자의 “질”이 다르기 때문이다. OpenAI의 적자는 소비자 Overhead + 인프라 투자가 혼합되어 있고, Anthropic의 적자는 R&D에 집중되어 있다. 매출 성장과 비용 구조의 관계가 다르기 때문에, 같은 적자라도 수익 전환의 경로가 다르다.

&circled6;
현금 · 재투자 — 순환은 지속되는가
CFS · FCF · CCC=DIO+DSO−DPO
OpenAI

IPO를 통한 자본 순환 설계. 현재 FCF는 대규모 음(-)이지만, PBC 전환을 완료하고 IPO를 준비 중이다. 상장을 통해 공개 시장에서 자본을 조달하고, $600B 인프라 투자를 위한 장기 자금을 확보하는 전략. 현금 소진 속도(burn rate)가 빨라 외부 자본 유입이 중단되면 리스크가 급격히 커진다.

FCF: 대규모 음(-) · IPO 준비 중 · 인프라 CapEx: $600B 계획
Anthropic

프라이빗 라운드로 자본 순환 유지. Series G($30B, 2026.02)까지 프라이빗으로 자본을 유치하며 순환을 이어가고 있다. API 종량제 구조는 매출 성장이 곧 현금 유입 개선과 연결되므로, 외부 자본 의존도를 점진적으로 줄여가는 경로가 존재한다.

FCF: 음(-), 개선 추세 · 최근 라운드: $30B (Series G) · 밸류: $380B
Node 6 통찰

AI 산업의 현금 순환은 “모델 훈련(대규모 CapEx) → 추론 서비스(매출) → 재훈련”의 사이클이다. OpenAI는 이 사이클을 IPO를 통한 공개시장 자본으로 돌리려 하고, Anthropic은 매출 성장 자체로 사이클을 자립시키려 한다. 장기적으로 어느 경로가 더 지속 가능한가는, 추론 비용이 얼마나 빨리 하락하느냐에 달려 있다 — 추론 비용이 떨어지면 API 종량제의 Anthropic이 더 빨리 자립한다.

OpenAI는 소비자 플랫폼 전략으로 800M 사용자를 확보하고 $20B 매출을 달성했다. 네트워크 효과와 브랜드 인지도가 핵심 자산이며, “ChatGPT = AI”라는 등식을 만들어냈다. 하지만 소비자 시장의 낮은 전환 비용과 $9B 적자 구조는 장기 지속 가능성에 물음표를 남긴다.

Anthropic은 엔터프라이즈 인프라 전략으로 32% 시장 점유율과 $9B 매출을 달성했다. API 전환 비용과 안전성 신뢰가 핵심 자산이며, 연 900% 성장률이 시장에서 $380B 밸류에이션을 정당화하고 있다. 하지만 모델 성능이 수렴하는 시장에서 기술적 차별화가 얼마나 지속될 수 있는지가 관건이다.

같은 지능을 만들지만, 하나는 넷플릭스가 되려 하고 다른 하나는 AWS가 되려 한다. 두 순환 구조 모두 $300B+ 기업을 만들어냈다는 점에서 “어느 쪽이 옳다”는 판정은 성립하지 않는다. 진짜 질문은 이것이다 —

AI의 추론 비용이 제로에 수렴하고,
에이전트가 인간의 업무를 대체하기 시작할 때,
“소비자 800M”의 플랫폼과 “기업 30만 개”의 인프라 중
어느 순환 구조가 먼저 자립하는가?
